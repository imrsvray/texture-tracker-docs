Detailed Description

Data structure based on a node with children, each of which are also Trees.

The Tree is the princliple class forming the datastructure. Within the node the tree can contain any objects as desired. It also has a fixed number of children that are also Trees (or null Trees). This can be any number, but it must be fixed throughout the wider Tree. (ie the root node and all its children)
