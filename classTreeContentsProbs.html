Detailed Description

Probabilites for a the features to have reached this leaf node.

This is a version of the TreeContents that goes inside leaf nodes of the tree. It contains details of which feature points have 'landed' at this point providing a method for calculating the accompanying probability.

void TreeContentsProbs::addFeature ( const int &  ID)

Recording that the feature reached this node.

When a feature is dropped into the tree it eventually reaches a leaf node. This function adds that feature to its records of which features have reached this leaf, creating an array of 'scores' for those features.
