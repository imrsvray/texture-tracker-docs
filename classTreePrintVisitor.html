Detailed Description

Class to traverse the tree and print its details to file.

An instance of this class is given the file pointer, it then traverses the whole tree, printing to file. Note: the structure is printed out in a rather verbose XML format

virtual int TreePrintVisitor::decisionVisit ( TreeContents &  Node) [virtual]

Function to determine how the Visitor should progress through the tree.

The visitor is controlled by the decisionTraversal function of the Tree. This implementation uses the 'compare points' function of the coordOffset testSubject held within the visitor.

Parameters:
NodeThe current internal node of the tree
Returns:
The value of the branch of the tree that should be followed next. i.e the index of the array of children. Result is -1 if no more progress is possible
