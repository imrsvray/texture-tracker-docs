Detailed Description

Class to traverse the tree and return with the results for a particular feature.

An instance of this class is given the details of a feature, it then traverses the tree in an appropriate manner for that feature, and when it reaches a leaf node it returns with a pointer to that node, which in turn contains details of the numbers and quantities of featues that reached that node during the training phase. Note: misleading name because all Visitors traverse trees.
