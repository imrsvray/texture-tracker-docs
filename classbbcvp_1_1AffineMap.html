Detailed Description

This class offers methods for performing affine transformations.

Uses Graham Thomas's c code, converted to a C++ class by Tim Sargeant. Important: This class uses the Solve class to solve it's equations.

Function to calculate mapping coefficients to map from coordinates of points on a plane, to where these points appear after a transformation equivalent to taking a photograph of the plane using a camera at an arbitrary place, with arbitrary pixel aspect ratio and focal length.

Uses this transformation:

 xout = (ax + by + c) / (gx + hy + 1)
 yout = (dx + ey + f) / (gx + hy + 1)

inputs: xi[ ], yi[ ] are four pairs of input coordinates xo[ ], yo[ ] are where these should map to

output: array coeffs[0..7] corresponding to a..h in the above mapping

Parameters:
xi (input) 4-length array of input x coordinates
yi (input) 4-length array of input y coordinates
xo (output) 4-length array of output x coordinates
yo (output) 4-length array of output y coordinates
coeffs 
Returns:
0 for success, -1 if equations can't be solved (coeffs[0..7] set to zero)
