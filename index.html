VPutil documentation

Introduction

This collection of C++ classes provide utility functions of general VP use. Candidiate classes might include useful mathematical functions, conversions, etc.

Details

This library is a recent creation and only has a few classes in so far! Please add some more soon... There are two useful functions for allocating and freeing n-dimensional arrays - see nd_array.h
